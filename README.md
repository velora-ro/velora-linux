# 🌲 Velora Linux

**Modern. Natural. Elegant.**

Velora Linux is a Debian-based Linux distribution built for everyday users, gamers, and creators.
Designed to be fast, beautiful, and Windows-compatible out of the box.

## Features

- 🖥️ KDE Plasma desktop with Velora theme
- 🎮 Gaming ready: Steam, Heroic, Lutris, Wine, Bottles, DXVK
- 🪟 Windows app compatibility via Wine + Bottles
- 📦 Flatpak + Flathub pre-configured
- 🛠️ Velora Hub - manage everything without the terminal
- 🌿 Inspired by nature

## Applications

| App | Description |
|-----|-------------|
| Velora Welcome | First-boot setup wizard |
| Velora Hub | System management center |
| Velora Settings | System settings |
| Velora Update | Update manager |
| Velora Driver Manager | GPU/hardware drivers |
| Velora Backup | System backup |
| Velora Store | App store |

## Build

```bash
# Install dependencies (Debian/Ubuntu)
sudo apt install live-build

# Clone the repo
git clone https://github.com/velora-ro/velora-linux
cd velora-linux

# Build the ISO
./build.sh
```

## Requirements

- Debian 12 or Ubuntu 22.04+ (or WSL2)
- 20GB free disk space
- Internet connection
